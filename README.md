# Calculator

Создать калькулятор для работы с рациональными и комплексными числами,
организовать меню,
добавить в систему логирования.

## Модули:
1 модуль - работа с комплексными числами 
2 модуль - работа с рациональными числами
3 модуль - управления
4 модуль - пользовательский интерфейс
5 модуль - логирование
